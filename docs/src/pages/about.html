About

@lopatnov/namespace

This is the documentation and live demo for the @lopatnov/namespace ecosystem — a collection of lightweight, tree-shakeable modules that complement jQuery and bring structure to web applications.

Every page you see here is a live demonstration of the namespace (core) and namespace-router (SPA routing with lazy loading). The pages are loaded on demand, route params work, and the sidebar highlights the current route — all powered by this library.
